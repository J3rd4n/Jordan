Kali Linux 模拟器 - 项目说明


项目特点


🖥️ 完整的桌面环境
• 启动画面 - 模拟Kali Linux启动过程，显示内核加载、服务启动等信息
• 登录界面 - 支持用户登录，包含用户名输入和密码验证（默认密码：kali）
• 桌面环境 - 完整的图形界面，包括顶部面板、应用程序菜单、底部Dock栏
• 窗口管理 - 支持窗口拖拽、最小化、最大化、关闭、Alt+F4快捷键


💻 终端模拟器
• 完整的Linux命令 - 支持 ls, cd, pwd, cat, touch, mkdir, rm, cp, mv 等文件操作命令
• 系统信息命令 - uname, whoami, hostname, date, df, free, ps, top 等系统管理命令
• 网络工具 - ifconfig, ping, netstat, nslookup, dig 等网络诊断工具
• 安全工具命令 - nmap（网络扫描）, whois（域名查询）等安全相关命令
• 命令历史 - 支持上下箭头浏览命令历史记录
• Tab补全 - 自动补全文件名和命令，提高操作效率
• 命令帮助 - man命令提供详细的使用说明


🗂️ 虚拟文件系统
• 完整的目录结构 - 模拟真实的Kali Linux文件系统，包含 /bin, /etc, /home, /usr 等标准目录
• 文件操作 - 创建、读取、写入、删除文件和目录，支持复制和移动操作
• 权限系统 - 模拟Linux文件权限（drwxr-xr-x），支持权限查看
• 特殊文件 - /proc（进程信息）, /dev（设备文件）, /sys（系统信息）等特殊目录
• 用户目录 - 完整的 /home/kali 用户目录结构，包含Desktop、Documents、Downloads等


🛠️ 安全工具应用
• Nmap - 网络扫描器，支持TCP/UDP扫描、服务检测
• Wireshark - 网络协议分析器，实时捕获和分析网络数据包
• Metasploit - 渗透测试框架，包含漏洞利用模块
• Burp Suite - Web应用安全测试工具，支持拦截和修改HTTP请求
• John the Ripper - 密码破解工具，支持字典攻击和暴力破解
• Hydra - 暴力破解工具，支持SSH、FTP等多种服务
• Aircrack-ng - 无线网络安全工具，支持WPA/WPA2破解
• Gobuster - 目录枚举工具，用于发现Web应用隐藏目录
• SQLMap - SQL注入工具，自动检测和利用SQL注入漏洞


🎨 视觉效果
• Kali Linux主题 - 深色主题，使用经典的Kali蓝色作为主色调
• 流畅动画 - 窗口打开/关闭动画、菜单过渡效果、按钮悬停效果
• 响应式设计 - 适配不同屏幕尺寸，保持良好的用户体验
• ASCII艺术 - 启动画面和欢迎信息包含精美的ASCII艺术
• 图标设计 - 统一的SVG图标风格，保持视觉一致性


使用方法


1. 打开项目
   • 双击打开 index.html 文件
   • 或在浏览器中打开 C:\Users\29869\kali-linux-simulator\index.html


2. 启动和登录
   • 等待启动画面完成（约4秒）
   • 在登录界面输入用户名：kali（已预填）
   • 输入密码：kali
   • 点击"Log In"按钮登录
   • 也可以选择"Guest Session"访客模式或"Terminal Only"纯终端模式


3. 使用桌面环境
   • 点击左上角"Applications"按钮打开应用程序菜单
   • 双击桌面图标启动应用程序
   • 点击底部Dock栏图标快速访问常用应用
   • 点击"Show Desktop"按钮最小化所有窗口


4. 使用终端
   • 从应用程序菜单或Dock栏启动终端
   • 输入命令并按回车执行
   • 使用上下箭头浏览命令历史
   • 使用Tab键自动补全
   • 输入"help"查看所有可用命令


5. 使用安全工具
   • 从应用程序菜单选择所需的安全工具
   • 配置扫描参数
   • 点击"Start"或"Scan"按钮执行
   • 查看扫描结果


常用命令示例
• ls -la              # 列出当前目录详细文件信息
• cd /home/kali        # 切换到用户主目录
• pwd                  # 显示当前工作目录
• cat readme.txt       # 查看文件内容
• mkdir newfolder      # 创建新目录
• touch newfile.txt    # 创建新文件
• rm oldfile.txt       # 删除文件
• cp source.txt dest.txt  # 复制文件
• mv oldname.txt newname.txt  # 重命名文件
• uname -a             # 显示系统信息
• ifconfig             # 显示网络接口信息
• ping 192.168.1.1     # Ping测试
• nmap 192.168.1.1     # 网络扫描

项目优势
• 无需安装 - 纯前端实现，直接在浏览器中运行
• 跨平台 - 支持Windows、Mac、Linux等所有现代浏览器
• 高度还原 - 完整模拟Kali Linux的命令行和图形界面
• 教育价值 - 适合学习Linux命令和安全工具使用
• 安全环境 - 完全沙箱化，不会影响真实系统


注意事项
• 这是一个模拟器，所有操作都在浏览器内存中进行
• 关闭浏览器后，所有修改将丢失
• 网络扫描等功能是模拟的，不会真正扫描网络
• 建议在桌面端浏览器中使用以获得最佳体验

• 无框架，纯原生实现
